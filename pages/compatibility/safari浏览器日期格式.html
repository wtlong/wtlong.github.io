# 传参NaN

safari 浏览器和 ios 嵌套页面都会遇到这个问题，会导致接口传的参数值为 NaN。

# 拼接格式

使用“/”，不能使用“-”，初始化和拼接时都必须要使用“/”。

# 转换格式

字符串拼接完成之后转换字符串格式。

var date = new Date('2018/8/24');
var year = date.getFullYear();
var month = date.getMonth()+1;
var day = date.getDate();
var startDate = year+"/"+month+"/"+day;//当前日期
var startDate = String(startDate);

var seconds = new Date(Date.parse(startDate)-24*60*60*1000*10);
var endYear = seconds.getFullYear();
var endMonth = seconds.getMonth()+1;
var endDay = seconds.getDate();
var endDate = endYear +"/"+endMonth+"/"+endDay;
var endDate = String(endDate);`
